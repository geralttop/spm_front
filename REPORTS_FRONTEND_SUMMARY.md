# Система жалоб - Фронтенд

## ✅ Что реализовано

### 1. API клиент для жалоб
- `src/shared/api/reports.ts` - API методы для работы с жалобами
- Типы данных для создания и получения жалоб
- Интеграция с существующим API клиентом

### 2. Компонент модального окна жалоб
- `src/shared/ui/report-modal.tsx` - универсальное модальное окно
- Поддержка жалоб на точки, комментарии и пользователей
- Валидация формы и обработка ошибок
- Мультиязычность (русский, английский, белорусский)

### 3. Интеграция в существующие компоненты

#### Точки (`src/shared/ui/point-card.tsx`)
- Кнопка жалобы рядом с кнопкой избранного
- Показывается только авторизованным пользователям
- Скрыта для собственных точек пользователя

#### Комментарии (`src/widgets/Comments/Comments.tsx`)
- Кнопка жалобы вместо кнопок редактирования для чужих комментариев
- Стилизация в соответствии с дизайном компонента
- Иконка флага с текстом "Пожаловаться"

#### Пользователи (`src/shared/ui/user-card.tsx`)
- Кнопка жалобы в карточке пользователя
- Опциональное отображение через пропс `showReportButton`
- Скрыта для собственного профиля

### 4. Система уведомлений
- `src/shared/ui/toast.tsx` - компонент тостов
- Хук `useToast` для управления уведомлениями
- Автоматическое исчезновение через 3 секунды
- Интеграция в layout wrapper

### 5. Переводы
- Добавлены переводы на 3 языка (ru, en, be)
- Ключи для всех элементов интерфейса жалоб
- Типы жалоб, причины, сообщения об ошибках

## 📱 Использование

### Жалоба на точку:
1. Найти точку в ленте или на карте
2. Нажать кнопку флага рядом с сердечком
3. Выбрать причину жалобы
4. Опционально добавить описание
5. Нажать "Отправить жалобу"

### Жалоба на комментарий:
1. Открыть комментарии к точке
2. Найти чужой комментарий
3. Нажать "Пожаловаться" под комментарием
4. Заполнить форму жалобы

### Жалоба на пользователя:
1. Найти пользователя в поиске
2. Нажать кнопку флага в карточке
3. Указать причину жалобы
