# Дашборд админки

## Обзор

Дашборд предоставляет визуальную статистику по всем данным в системе.

## Компоненты дашборда

### 1. Карточки с общей статистикой

Четыре карточки показывают основные метрики:
- **Пользователи** - общее количество зарегистрированных пользователей
- **Точки** - общее количество созданных точек на карте
- **Категории** - количество категорий (системных и пользовательских)
- **Контейнеры** - количество контейнеров для группировки точек

### 2. Графики

#### Распределение по ролям (Pie Chart)
Показывает соотношение администраторов и обычных пользователей в системе.

#### Подтверждение Email (Pie Chart)
Визуализирует, сколько пользователей подтвердили свой email.

#### Точки по категориям (Bar Chart)
Столбчатая диаграмма показывает распределение точек по категориям.
- Каждый столбец окрашен в цвет соответствующей категории
- Помогает понять, какие категории наиболее популярны

#### Топ-10 авторов точек (Horizontal Bar Chart)
Горизонтальная столбчатая диаграмма показывает самых активных пользователей.
- Отображает 10 пользователей, создавших больше всего точек
- Помогает выявить самых активных участников

#### Создание точек за последние 30 дней (Line Chart)
Линейный график показывает динамику создания точек.
- Отображает количество созданных точек по дням
- Помогает отследить активность пользователей во времени
- Показывает тренды и пики активности

## API Endpoint

Дашборд получает данные из эндпоинта:
```
GET /admin/stats
```

## Технологии

- **Recharts** - библиотека для создания графиков
- **React Admin Cards** - карточки для отображения метрик
- **Responsive Design** - адаптивная сетка для разных экранов

## Обновление данных

Данные загружаются при открытии дашборда. Для обновления:
1. Перезагрузите страницу
2. Или перейдите на другую вкладку и вернитесь обратно

В будущем можно добавить:
- Автоматическое обновление каждые N секунд
- Кнопку "Обновить"
- WebSocket для real-time обновлений
