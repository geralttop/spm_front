# Возможности админки SPM

## 📊 Дашборд

При входе в админку вы попадаете на дашборд с визуальной статистикой:

### Карточки метрик
- **Пользователи** - общее количество (синий цвет)
- **Точки** - количество точек на карте (зеленый цвет)
- **Категории** - количество категорий (желтый цвет)
- **Контейнеры** - количество контейнеров (фиолетовый цвет)

### Графики

1. **Распределение по ролям** (круговая диаграмма)
   - Показывает соотношение администраторов и обычных пользователей
   - Процентное соотношение

2. **Подтверждение Email** (круговая диаграмма)
   - Сколько пользователей подтвердили email
   - Сколько еще не подтвердили

3. **Точки по категориям** (столбчатая диаграмма)
   - Каждая категория - отдельный столбец
   - Цвет столбца соответствует цвету категории
   - Показывает популярность категорий

4. **Топ-10 авторов** (горизонтальная столбчатая диаграмма)
   - Самые активные пользователи
   - Количество созданных точек
   - Помогает выявить лидеров сообщества

5. **Динамика создания точек** (линейный график)
   - Последние 30 дней
   - Показывает тренды активности
   - Помогает понять, когда пользователи наиболее активны

## 👥 Управление пользователями

### Список пользователей
- Таблица со всеми пользователями
- Колонки: ID, имя, email, роль, статус подтверждения
- Пагинация (10/25/50/100 записей на странице)
- Сортировка по любой колонке
- Поиск и фильтрация

### Редактирование пользователя
- Изменение имени пользователя
- Изменение email
- Изменение роли (user/admin)
- Изменение статуса подтверждения email
- Редактирование биографии

### Просмотр пользователя
- Полная информация о пользователе
- Даты создания и обновления
- Все поля профиля

### Удаление пользователя
- Удаление с подтверждением
- Каскадное удаление связанных данных

## 📍 Управление точками

### Список точек
- Таблица со всеми точками
- Колонки: ID, название, адрес, автор, категория, контейнер, дата
- Пагинация и сортировка
- Фильтрация по автору, категории, контейнеру

### Создание точки
- Название и описание
- Адрес
- Координаты (долгота и широта)
- Выбор категории
- Выбор контейнера

### Редактирование точки
- Изменение всех полей
- Перемещение на карте (изменение координат)
- Смена категории или контейнера

### Просмотр точки
- Полная информация
- Координаты
- Связанные объекты (автор, категория, контейнер)

## 🏷️ Управление категориями

### Список категорий
- Все категории (системные и пользовательские)
- Название, иконка, цвет, автор
- Пагинация и сортировка

### Создание категории
- Название
- Иконка (название иконки)
- Цвет (hex код, например #FF0000)

### Редактирование категории
- Изменение названия
- Изменение иконки
- Изменение цвета

### Удаление категории
- Удаление с подтверждением
- Внимание: точки с этой категорией останутся без категории

## 📦 Управление контейнерами

### Список контейнеров
- Все контейнеры
- Название, описание, автор, дата создания
- Пагинация и сортировка

### Создание контейнера
- Название (title)
- Описание

### Редактирование контейнера
- Изменение названия
- Изменение описания

### Удаление контейнера
- Удаление с подтверждением
- ВНИМАНИЕ: Каскадное удаление всех точек в контейнере!

## 🔐 Безопасность

- Все операции требуют JWT токен
- Проверка роли admin на каждом запросе
- Автоматическое обновление токенов
- Защита от CSRF через httpOnly cookies
- Логирование всех действий администратора

## 🎨 Интерфейс

- Адаптивный дизайн (работает на всех устройствах)
- Темная/светлая тема (наследуется от основного приложения)
- Русский язык интерфейса
- Интуитивная навигация
- Быстрый поиск и фильтрация
- Экспорт данных (будет добавлено)
