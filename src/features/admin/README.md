# Админка SPM

Админка создана с использованием react-admin и доступна только пользователям с ролью `admin`.

## Доступ

1. Войдите в систему под учетной записью с ролью `admin`
2. В сайдбаре появится пункт "Админка"
3. Перейдите по адресу `/admin`

## Функциональность

### Управление пользователями
- Просмотр списка всех пользователей
- Редактирование профилей пользователей
- Изменение ролей пользователей
- Удаление пользователей

### Управление точками
- Просмотр всех точек в системе
- Создание новых точек
- Редактирование существующих точек
- Удаление точек

### Управление категориями
- Просмотр всех категорий
- Создание новых категорий
- Редактирование категорий
- Удаление категорий

### Управление контейнерами
- Просмотр всех контейнеров
- Создание новых контейнеров
- Редактирование контейнеров
- Удаление контейнеров

## API Endpoints

Все админские операции выполняются через `/admin/*` endpoints:

- `GET /admin/users` - список пользователей
- `GET /admin/points` - список точек
- `GET /admin/categories` - список категорий
- `GET /admin/containers` - список контейнеров

И соответствующие CRUD операции для каждого ресурса.

## Безопасность

- Доступ к админке защищен JWT токенами
- Проверяется роль пользователя на уровне API
- Все операции логируются
