# Widgets

Компоненты верхнего уровня для структуры приложения.

## LayoutWrapper

Условный рендеринг layout в зависимости от текущего пути:

- **Обычные страницы** (`/feed`, `/profile`, и т.д.) - показывает Sidebar + Header
- **Страница авторизации** (`/auth`) - чистая страница без навигации
- **Админка** (`/admin`) - чистая страница для react-admin интерфейса

Это позволяет админке использовать собственный layout от react-admin без конфликтов с основным layout приложения.
