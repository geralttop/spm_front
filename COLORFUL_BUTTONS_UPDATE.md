# Обновление кнопок подписки - добавление цветов

## Изменения
Все кнопки подписки/отписки теперь имеют яркие цвета для лучшего UX.

## Цветовая схема

### Кнопка "Подписаться"
- **Цвет**: Синий (`bg-blue-500`)
- **Hover**: Темно-синий (`hover:bg-blue-600`)
- **Текст**: Белый (`text-white`)
- **Граница**: Синяя (`border-blue-500`)

### Кнопка "Отписаться"
- **Цвет**: Красный (`bg-red-500`)
- **Hover**: Темно-красный (`hover:bg-red-600`)
- **Текст**: Белый (`text-white`)
- **Граница**: Красная (`border-red-500`)

## Обновленные компоненты

### 1. Страница профиля (`/profile`)
- **Модальное окно подписчиков**: 
  - Синие кнопки "Подписаться" для новых подписок
  - Красные кнопки "Отписаться" для существующих подписок
- **Модальное окно подписок**:
  - Красные кнопки "Отписаться" для всех пользователей

### 2. Страница пользователя (`/user/[id]`)
- **Основная кнопка подписки**:
  - Синяя кнопка "Подписаться" если не подписан
  - Красная кнопка "Отписаться" если подписан

## Визуальные улучшения
- Кнопки теперь более заметные и интуитивно понятные
- Красный цвет ассоциируется с удалением/отпиской
- Синий цвет ассоциируется с добавлением/подпиской
- Белый текст обеспечивает хорошую контрастность
- Hover эффекты делают кнопки более интерактивными

## Техническая реализация
Используются Tailwind CSS классы с кастомными стилями через `className` prop для переопределения стандартных вариантов Button компонента.
