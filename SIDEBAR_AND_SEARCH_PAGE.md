# Добавление бокового меню и отдельной страницы поиска

## Новая функциональность

### 1. Боковое меню (Sidebar)
Создано фиксированное боковое меню с навигацией по основным разделам приложения.

#### Компоненты:
- `src/widgets/sidebar/sidebar.tsx` - основной компонент сайдбара
- `src/widgets/sidebar/index.ts` - экспорт компонента

#### Особенности:
- **Фиксированное позиционирование** - остается на месте при скролле
- **Адаптивная подсветка** - активный пункт меню подсвечивается
- **Условное отображение** - скрывается на странице авторизации и для неавторизованных пользователей
- **Иконки** - каждый пункт меню имеет соответствующую иконку из Lucide React

#### Пункты меню:
- 🏠 **Главная** (`/`) - домашняя страница
- 👤 **Профиль** (`/profile`) - страница профиля пользователя
- 🔍 **Поиск** (`/search`) - отдельная страница поиска
- 🗺️ **Карта** (`/map`) - будущая страница с картой
- 👥 **Сообщество** (`/community`) - будущая страница сообщества
- 🔔 **Уведомления** (`/notifications`) - будущая страница уведомлений
- ⚙️ **Настройки** (`/settings`) - будущая страница настроек

### 2. Отдельная страница поиска
Создана полноценная страница поиска пользователей с улучшенным UX.

#### Компонент:
- `app/search/page.tsx` - страница поиска

#### Особенности:
- **Полноэкранный интерфейс** - больше места для результатов
- **Улучшенный дизайн** - карточки пользователей с аватарами
- **Состояния интерфейса**:
  - Начальное состояние с подсказкой
  - Состояние поиска с индикатором загрузки
  - Результаты поиска с количеством найденных пользователей
  - Пустые результаты с предложением изменить запрос
- **Фильтрация** - исключает текущего пользователя из результатов
- **Навигация** - клик по карточке ведет на профиль пользователя

### 3. Обновленный Layout
Изменена структура приложения для размещения сайдбара.

#### Изменения:
- Добавлен импорт `Sidebar` компонента
- Изменена структура layout:
  ```tsx
  <div className="flex min-h-screen">
    <Sidebar />
    <div className="flex-1 ml-64">
      <Header />
      <main>{children}</main>
    </div>
  </div>
  ```
- Основной контент сдвинут на 256px (64 * 4) вправо

### 4. Упрощенный Header
Убрана функциональность поиска из header, так как теперь есть отдельная страница.

#### Изменения:
- Удален поиск и связанные состояния
- Упрощены импорты
- Оставлены только логотип и переключатели языка/темы

### 5. Переводы
Добавлены переводы для новых элементов интерфейса.

#### Новые ключи:
- `sidebar.*` - названия пунктов меню
- `search.*` - тексты для страницы поиска

#### Языки:
- ✅ Английский (`en/common.json`)
- ✅ Русский (`ru/common.json`)

## Техническая реализация

### Стили и позиционирование:
- Сайдбар: `fixed left-0 top-0 h-full w-64`
- Основной контент: `flex-1 ml-64`
- Z-index: сайдбар имеет `z-40`, header - `z-50`

### Состояния и логика:
- Проверка аутентификации для отображения сайдбара
- Активная подсветка на основе `usePathname()`
- Фильтрация результатов поиска

### Адаптивность:
- Фиксированная ширина сайдбара (256px)
- Контент адаптируется к оставшемуся пространству
- Готовность к будущим мобильным адаптациям
