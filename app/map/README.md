# Страница карты

Полноэкранная карта с отображением всех точек из ленты подписок пользователя.

## Функциональность

- Отображение всех точек пользователей, на которых подписан текущий юзер
- Интерактивные маркеры с цветами категорий
- Popup с подробной информацией о точке при клике на маркер
- Переход на страницу точки из popup
- Выбор стиля карты (OpenStreetMap, Satellite, Carto и др.)
- Элементы управления картой (зум, компас, геолокация, полноэкранный режим)
- Счетчик отображаемых точек
- Индикатор загрузки
- Обработка ошибок

## Технологии

- Next.js 14 (App Router)
- MapLibre GL JS (через компонент Map)
- React Query для загрузки данных
- Zustand для управления состоянием
- i18next для интернационализации

## API

Использует endpoint `/points/feed` для получения точек из подписок.

## Навигация

Страница доступна по пути `/map` и добавлена в:
- Боковое меню (Sidebar)
- Мобильную навигацию (MobileNav)

## Особенности

- Страница отображается на весь экран без header и sidebar
- Автоматическое центрирование на первой точке при загрузке
- Адаптивный дизайн для мобильных устройств
- Поддержка темной и светлой темы
